2) Команда git add приймає шлях файлу або директорії. Якщо це директорія, команда додає усі файли в цій директорії рекурсивно.
3) git status показує список файлів.
4) git commit бере всі дані, додані до індексу, і зберігає їх зліпок у внутрішній базі даних, а потім зрушує покажчик поточної гілки на цей зліпок.
5) коли ви пишете "git init", Git "включається" або "запускається" для даного репозиторію.
